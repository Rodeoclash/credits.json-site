Credits.json

A machine readable data format that describes the creators of a website.

Isn't this humans.txt?

Humans.txt is is not designed to be machine parsable, if it was it would defeat its original purpose. credits.json is designed to sit alongside a humants.txt and provide the same information in a format that machines can understand.

Why?

It makes it possible to write tools that can automatically analyse and track who worked on websites across the Internet.
